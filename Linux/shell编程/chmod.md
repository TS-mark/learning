# chmod

---

`chmod`是用于改变文件权限的命令，是"change mode"的缩写。

该命令允许修改文件的权限，基本语法如下：
基本语法:
```bash
chmod [选项] 模式 文件名
```

```bash
chmod 755 file.txt
```
将`file.txt`设置为所有者具有读、写、执行权限，其他用户具有读和执行权限。  

1. 数字模式（通常使用）：  
在数字模式中，每个权限用一个数字表示，组合起来构成一个三位数。每个数字代表一组权限：所有者、所属组、其他用户。  
数字模式中每个数字的含义：
* 0：没有任何权限。
* 1：执行权限。
* 2：写权限。
* 3：写和执行权限。
* 4：读权限。
* 5：读和执行权限。
* 6：读和写权限。
* 7：读、写和执行权限。

2. 符号模式：
* `u` 所有者
* `g` 所属组
* `o` 其他用户
* `a` 所有用户

加号和减号：  
* `+`增加权限  
* `-`移除权限  

符号：  
* `r`读权限    
* `w`写权限  
* `x`执行权限
